#### Automatically use nearby Boss Replay machines

Auto Encore will automatically trigger a nearby Boss Replay machine when you are within the configured range.

**Active :** Enables automatic Boss Replay machine use

**Inactive :** Disables automatic Boss Replay machine use

#### Settings

- Auto Use Radius (m)
  - Sets the distance around your character where Boss Replay machines can be auto-used
- Use Type
  - 0 = Primary use (money)
  - 1 = Secondary use (eridium)

#### Notes

- The mod checks for nearby Boss Replay machines while you are in-game
- If multiple machines are within range, the nearest one will be used
